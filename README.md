# Лендинг «Траст Лидс»

Одностраничный продающий лендинг (CPA-маркетинг). Реализован по дизайн-референсу
из Claude Design проекта «Траст Лидс лендинг» — один самодостаточный файл
`index.html` на чистом HTML/CSS/JS, без сборки и зависимостей.

## Как открыть
Просто двойной клик по `index.html` — откроется в браузере. Интернет нужен только
для подгрузки шрифтов Google Fonts (можно работать и без них — будет системный шрифт).

## Что нужно заполнить (плейсхолдеры)
Все ссылки и тексты вынесены в блок **`CONFIG`** в начале `<script>` внизу файла.
Пустая строка `""` оставляет плейсхолдер, а ссылку/кнопку делает неактивной.

| Поле | Что вписать |
|---|---|
| `vkUrl` | Ссылка на сообщество во ВКонтакте |
| `ofertaUrl` | Путь к PDF оферты в `web/` (напр. `oferta-trustleads.pdf`). Ссылка «ОФЕРТА» открывает его, а в модалке он встраивается для чтения. |
| `ofertaText` | Текст оферты (HTML). Если задан — показывается ВМЕСТО встроенного PDF |
| `instructionUrl` | Ссылка на инструкцию по регистрации ИП |
| `video1Url`, `video2Url` | Видео. Поддерживаются два варианта (см. ниже) |
| `metrikaId` | Номер счётчика Яндекс.Метрики (только цифры). Пусто — аналитика выключена (см. ниже) |
| `telegramUrl`, `maxUrl` | Ссылки на мессенджеры под QR-кодами (блок «Остались вопросы?») |
| `services[]` | сервисы регистрации ИП: `name`, `desc`, `url`, `tariff`, `mode` |
| `reviews[]` | отзывы участников: `name`, `meta`, `stars`, `text`, `avatar` |
| `faq[]` | вопросы-ответы (аккордеон): объекты `{ q, a }` |
| `showStars`, `parallax`, `starDensity` | Настройки космического фона |

## Видео
Поле `video1Url` / `video2Url` понимает два формата:
- **Локальный файл** — положите `.mp4` (или `.webm`/`.mov`) в папку `web/video/`
  и укажите относительный путь, напр. `"video/cpa-trustleads.mp4"`. Покажется
  встроенный HTML5-плеер с управлением. *(Видео 1 уже подключено так.)*
- **Встраивание** — embed-ссылка, напр. YouTube `"https://www.youtube.com/embed/XXXX"`.

Пусто `""` — останется плейсхолдер с кнопкой play.

## Яндекс.Метрика (счётчик посетителей и просмотров видео)
Аналог Яндекс.Метрики уже встроен — нужно только подключить свой счётчик.

**Шаг 1. Завести счётчик.**
1. Откройте https://metrika.yandex.ru → войдите под Яндекс-аккаунтом → «Добавить счётчик».
2. Имя — любое (напр. «Траст Лидс лендинг»), сайт — `start.trust-leads.ru`.
3. Включите «Вебвизор» (запись действий) — по желанию. Сохраните.
4. Скопируйте **номер счётчика** — это 8 цифр (напр. `99887766`), не путать с кодом.

**Шаг 2. Включить на сайте.**
Впишите этот номер в `CONFIG.metrikaId` в `index.html`, напр. `metrikaId: "99887766"`.
Готово — Метрика начнёт считать всех посетителей сайта автоматически.

**Шаг 3. Счётчик именно на ВТОРОЕ видео.**
Просмотр (запуск) второго видео «Как начать?» страница сама отправляет в Метрику
как **цель** с идентификатором `video2_play`. Чтобы увидеть число:
1. В кабинете Метрики → «Настройка» → вкладка **«Цели»** → «Добавить цель».
2. Тип условия — **«JavaScript-событие»**, идентификатор цели — `video2_play`.
3. Название — напр. «Запуск 2-го видео». Сохраните.

После этого в отчётах Метрики (раздел «Отчёты → Конверсии», либо «Цели») будет видно,
сколько человек запустили второе видео. Аналогично заведена цель `video1_play` для
первого видео — если захотите сравнить.

> Цель засчитывается один раз на загрузку страницы (повторный запуск после паузы
> число не накручивает). Данные в отчётах появляются с задержкой в несколько минут.

## Сервисы регистрации ИП
Список в `CONFIG.services`. У каждого сервиса есть `mode`:
- `"link"` — кнопка «Открыть» ведёт по `url` (Озон Банк, СберБанк — уже с реф-ссылками).
- `"form"` — кнопка «Оформить заявку» открывает форму на сайте (ВТБ — реф-ссылки нет,
  заявки собираются и подаются вручную через агента).
- `"disabled"` — кнопка показывает сообщение `message` (Альфа-Банк — выездной сервис
  временно приостановлен).

## Форма заявки ВТБ → Google-таблица
Форма (ФИО, телефон, e-mail, город + согласие на обработку ПД)
отправляет заявки в вашу Google-таблицу. Чтобы это заработало:

1. Откройте файл `web/google-apps-script.gs` и выполните инструкцию в его шапке
   (создать таблицу → Apps Script → вставить код → развернуть как веб-приложение).
2. Скопируйте полученный URL веб-приложения (оканчивается на `/exec`).
3. Вставьте его в `CONFIG.vtbFormEndpoint` в `index.html`.

Пока `vtbFormEndpoint` пуст — форма открывается, но при отправке пишет, что приём заявок
ещё не подключён. После вставки URL заявки начнут падать строками в таблицу.

## Блок «Остались вопросы?» (контакты)
Три QR-кода (Telegram, MAX, ВКонтакте) + кнопки. Положите файлы QR в папку `web/`
с именами:
- `qr-telegram.png` — QR Telegram (@TRUSTLEADSHUB)
- `qr-max.png` — QR MAX (Trust Leads)
- `qr-vk.png` — QR ВКонтакте (vk.ru/trustleadscompany)

Пока файлов нет — показывается заглушка «QR-код появится здесь». Ссылки кнопок —
в `CONFIG.telegramUrl` и `CONFIG.maxUrl`.

## Логика модалки «Присоединиться»
1. Чек-бокс — акцепт оферты. Пока не отмечен, выбор сервиса заблокирован (размыт).
2. После отметки разблокируется список сервисов регистрации ИП.
3. Внизу — кнопка перехода в сообщество ВКонтакте.

## Деплой на сервер (Beget VDS)
Лендинг статический, отдаётся любым веб-сервером. Пример для nginx:

```nginx
server {
    listen 80;
    server_name ВАШ_ДОМЕН;
    root /opt/trustleads_bot/web;
    index index.html;
    location / { try_files $uri $uri/ /index.html; }
}
```

Залить папку `web/` на сервер, прописать конфиг, `nginx -t && systemctl reload nginx`.
Для HTTPS — выпустить сертификат через `certbot`.
